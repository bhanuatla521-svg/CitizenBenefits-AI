# CitizenBenefits AI 🏛️

An intelligent AI-powered platform to help citizens discover government schemes and benefits they're eligible for.

## 🎯 Problem Statement
Millions of citizens don't know which government schemes they're eligible for, leading to missed opportunities for financial aid, education, healthcare, and social support.

## ✨ Features

### Core Features
- **Smart Eligibility Assessment** - Age, Income, State, Occupation analysis
- **Personalized Recommendations** - AI-powered scheme matching
- **Multi-Scheme Coverage:**
  - 🎓 Scholarships & Education Benefits
  - 👨‍🌾 Farmer Schemes & Agricultural Support
  - 👴 Pension & Senior Citizen Programs
  - 🏥 Health Insurance & Medical Benefits
  - 💼 Business Loans & Entrepreneurship
  - 👨‍👩‍👧‍👦 Family & Child Support
  - 🏠 Housing & Rental Schemes
  - ⚡ Employment & Skill Training
  - 🛡️ Social Security & Insurance

### Enhanced Features
- 📍 **Location-Based Filtering** - State and region-specific schemes
- 💾 **Save & Bookmark** - Keep track of favorite schemes
- 📊 **Application Tracking** - Monitor scheme applications
- 🔔 **Smart Notifications** - Deadline reminders and new scheme alerts
- 🌍 **Multi-Language Support** - Hindi, English, Regional languages
- 📱 **Mobile Responsive** - Access on any device
- 🔐 **Secure Data** - Privacy-first approach
- 💬 **Live Chat Support** - Real-time assistance
- 📄 **Document Upload** - Easy application process
- 🎯 **Progress Dashboard** - View all benefits at a glance

## 📊 Revenue Model
- **Affiliate Commissions** - Partner with financial institutions
- **Premium Consultation** - Expert guidance services
- **API Access** - B2B integration for other platforms
- **Sponsored Content** - Government agency promotions
- **Lead Generation** - Refer to partner services

## 🚀 Getting Started

### Prerequisites
- Node.js v16+
- MongoDB/PostgreSQL
- Docker (optional)

### Installation

```bash
git clone https://github.com/bhanuatla521-svg/CitizenBenefits-AI.git
cd CitizenBenefits-AI

# Backend setup
cd backend
npm install

# Frontend setup
cd ../frontend
npm install
```
